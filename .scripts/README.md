 - [findf](findf)
   - Search fname `$1` in dir `$2` with maxdepth `$3`

 - [findw](findw)
   - Search word `$1` in file `$2`
   - `${@:3}` additional params, e.g. ignore case: `-i`
 - [sound](sound)
